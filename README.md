# 🎓 The Complete Full-Stack Web Development Bootcamp

This repository contains all my **Capstone Projects** from
***The Complete Full-Stack Web Development Bootcamp*** by **Dr. Angela Yu** on **Udemy**.

Each capstone project is built **independently from scratch**, applying concepts learned throughout the course — covering everything from frontend design to backend logic and full-stack integration.

---

## 📁 Projects Included

### 🧑‍💼 Capstone Project: Resume

A personal **resume website** built with pure **HTML** and **CSS**.
Focuses on semantic structure, responsive layout, and professional presentation.

---

### 🌐 Capstone Project: Personal Website

A **multi-page personal website** showcasing skills, hobbies, and contact information.
Built using **HTML**, **CSS**, and basic **JavaScript** for interactivity.

---

### 📝 Capstone Project: Blog Web App

A **Node.js + Express.js** web app using **EJS** templating.
Users can create, read, and manage blog posts dynamically.

---

### 💹 Capstone Project: Stock Market

A web app that fetches and displays **real-time stock data** using APIs.
Focus on **API integration**, **data visualization**, and **asynchronous JavaScript**.

---

### 📚 Capstone Project: Book Notes

A **book note management app** to add, edit, and delete reading notes.
Explores **CRUD operations**, **data persistence**, and clean UI design.

---

## 🛠️ Tech Stack Highlights

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Templating:** EJS
* **APIs & Data Handling:** RESTful APIs, JSON
* **Tools:** Git, GitHub, VS Code
